# Nova Link Trademark Guidelines

Last updated: January 2026

## Overview

While the Nova Link client source code is licensed under the Apache License 2.0, the **Nova Link name, logo, and brand assets are NOT covered by that license**. They remain under exclusive ownership and control.

## Trademarks

The following are trademarks owned by the Nova Link Team:

- **"Nova Link"** (word mark)
- **The Nova Link logo** (stylized icon)
- **The Nova Link visual identity** (color scheme, UI design, branding elements)

## What You May NOT Do

Without explicit written permission, you may **NOT**:

1. **Use the name "Nova Link"** or any confusingly similar name for your fork, derivative work, or competing product
2. **Use the Nova Link logo or icon** in any fork, derivative work, or other project
3. **Imply endorsement or affiliation** with Nova Link or its team
4. **Create products or services** that appear to be official Nova Link offerings
5. **Use Nova Link branding** in domain names, social media handles, or app store listings

## What You MAY Do

You **MAY**:

1. Use the Nova Link name to accurately describe the origin of forked code (e.g., "Based on Nova Link" or "Fork of Nova Link")
2. Link to the official Nova Link repository or website
3. Make factual statements about Nova Link (e.g., reviews, comparisons)
4. Use the Apache-2.0 licensed source code under the terms of that license

## Fork Guidelines

If you create a fork of Nova Link:

1. You **MUST** choose a different name for your fork
2. You **MUST** create your own logo and branding
3. You **MUST** remove all Nova Link branding from your fork
4. You **SHOULD** clearly indicate that your project is a fork
